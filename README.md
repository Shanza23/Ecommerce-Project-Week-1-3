# Ecommerce Web Design — Frontend Development Task

A responsive ecommerce website built as part of the **DevelopersHub Frontend Development Internship** (Weeks 1–3), based on a Figma community design ("Ecommerce Web Design").

## 🔗 Live Pages

- `index.html` — Homepage
- `pages/products.html` — Product listing page (grid view with filters)
- `pages/product-detail.html` — Single product detail page

## Task Breakdown

**Week 1 — Header & Footer**
- Responsive header with logo, search bar, category dropdown, and account icons (Profile, Message, Orders, Cart)
- Navigation bar with category menu
- Footer with company info, link columns, newsletter signup, and supplier regions

**Week 2 — Homepage & Product Listing**
- Hero banner with category sidebar and promotional cards
- Deals & offers sections with countdown timer
- Recommended products grid
- Product listing page with sidebar filters (Category, Brands, Features, Price range, Condition, Ratings)
- Active filter chips with add/remove interactivity
- Grid/list view toggle

**Week 3 — Product Detail Page & Interactivity**
- Image gallery with clickable thumbnails
- Size/color/quantity selectors
- Tabbed content (Description, Specifications, Reviews)
- Add to Cart / Wishlist interactivity with toast notifications
- Related products section

## Built With

- HTML5
- CSS3 (custom properties, Flexbox, Grid)
- Vanilla JavaScript (no frameworks)
